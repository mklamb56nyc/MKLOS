# packages — monthly Received Wisdom output

One directory per month: `packages/YYYY-MM/` containing
- `expenses.xlsx` (or `.csv`) — that month's `rw` ledger rows, itemized
- copies of the month's `rw` receipts
- `summary.md` — totals by category, notes, anything the CPA should know

Each package pairs with a Gmail draft to the CPA (never auto-sent).
